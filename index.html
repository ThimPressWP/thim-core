What does Thim Core do!

  • Register necessary custom post types, custom fields for the themes.
  • Add a Control Panel, Customization Page to easier manage the theme.
  • Works as a One-Click Installer to install WordPress themes easier.

Installation

The latest version of the Thim Core plugin can be downloaded from the gh-pages branch as a WordPress installable package. DO NOT click the Github "Download ZIP" button, this plugin is meant to be installed in the WordPress plugins directory as thim-core and Github will not package the plugin correctly.

Installing the Thim Core plugin.

  1. Download the thim-core.zip and unpack the archive.
  2. Upload thim-core to the /wp-content/plugins/ directory.
  3. Activate the plugin through the 'Plugins' menu in WordPress.
  4. Read our guide on how to Get Started with Thim Core.

Thim Core Installer Configuration

  1. Download and include into theme folder.
  2. Add a filter: 
    add_filter( 'thim_core_installer_theme_config', function () {
    return array(
        'name'          => __( 'Example Theme', 'thim-text-domain' ),
        'slug'          => 'example-theme',
        'support_link'  => 'https://thimpress.com',
        'installer_uri' => get_template_directory_uri() . '/inc/admin/thim-core-installer'//Installer directory URI
    );
} );
